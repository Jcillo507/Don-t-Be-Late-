# Don't-Be-Late-
Don't be late is a racing obstacle avoidance game

You woke up late and have one minute to get there. Can you make it?
**Don't Be Late**

##Instructions
Avoid the oncoming traffic

Use the arrow keys to move

↑ move foward
↓ move back
← move left
→ move right
